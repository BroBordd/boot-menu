Features

Everything you need at boot time

⚡

Pre-framework

Launches before Android starts. Full control over the boot process from the very beginning.

🖥

Native OpenGL ES 2.0

60fps fullscreen overlay via SurfaceFlinger. No JVM, no Activity, no framework overhead.

👆

Touch & hardware keys

Raw evdev input. Volume keys to navigate, power to confirm. Full touch support.

🔄

Boot actions

System, reboot, recovery, download mode, power off — all with confirmation dialogs.

📊

Boot status bar

Animated progress bar while Android boots in background. Solid green when ready.

🧩

Extras

Drop any Stratum-based binary into extras/ and it appears in the Advanced menu.

Installation

How to install

1

Download the zip

Grab the zip for your device from the downloads section above.

2

Flash via KernelSU

Open KernelSU Manager → Modules → install from storage → select the zip.

3

Reboot

The boot menu appears on next startup before Android loads.

Development

Add your device

boot-menu is device-specific. Adding support for a new device requires forking Stratum and providing a device config.

# 1. clone with submodules git clone --recurse-submodules https://github.com/BroBordd/boot-menu cd boot-menu # 2. fork stratum and add your device # create stratum/devices/<model>/StratumConfig.h # create stratum/devices/<model>/app.cpp (copy from src/default.cpp) # create stratum/devices/<model>/stratum-boot/ (module source) # 3. build bash scripts/build.sh <model> # 4. output zip # stratum/devices/<model>/out/<model>-boot-menu.zip
